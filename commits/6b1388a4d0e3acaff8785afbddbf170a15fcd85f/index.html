Integrate with GitHub Actions

Declared a .github/workflows/git-read.yml file to configure steps necessary to leverage GitHub actions.

The desired outcome is to publish a git-read-generated website to that project’s [gh-pages branch].

To achieve this, execute the following steps upon a git push origin/master:

  • clone the host project’s entire source code history with fetch-depth: 0 to

  • install Ruby into the Action’s container

  • clone and install the git-read project into the container

    • execute bin/git-read to generate a website from the host project’s Git History

  • deploy the output to the host project’s gh-pages branch

Installation Script

Introduce the script/install file to execute the necessary commands to configure the project to be executable. Mainly, install its RubyGems dependencies.

+name: git-read
+
+on:
+  push:
+    branches:
+      - master
+
+jobs:
+  build:
+    runs-on: ubuntu-latest
+    steps:
+      - uses: actions/checkout@v2
+        with:
+          fetch-depth: 0
+
+      - uses: actions/cache@v1
+        with:
+          key: ${{ runner.os }}-gems-${{ hashFiles('**/Gemfile.lock') }}
+          restore-keys: ${{ runner.os }}-gems-
+          path: vendor/bundle
+
+      - uses: actions/setup-ruby@v1
+        with:
+          ruby-version: "2.6.5"
+          run: bundle config path vendor/bundle
+
+      - name: Build documentation from Git History
+        run: |
+          script/install
+          bin/git-read --git-dir ./ --build-dir ./build
+
+      - name: Deploy
+        uses: peaceiris/actions-gh-pages@v3
+        with:
+          personal_token: ${{ secrets.PERSONAL_TOKEN }}
+          commit_message: ${{ github.event.head_commit.message }}
+          publish_dir: ./build
