Syntax highlight Markdown fenced code blocks

Include the [syntax highlighting][syntax] CSS in source/stylesheets/syntax.css.erb.

The stylesheet is wholly generated from the rouge gem’s Github theme.

To enable it, configure Middleman’s Markdown rendering engine by adding a call to activate :syntax to the project’s config.rb.
