Restore default ul and ol within Markdown

Render commit messages that contain Markdown lists as the browser defaults.

For instance, a commit with an <ul> element’s items will be declared as:

  • item one
  • item two
  • items three

Whereas an <ol> element’s items will be declared as:

  1. item one
  2. item two
  3. item three

These styles were originally overridden in the base/base SCSS generated while installing bitters.
